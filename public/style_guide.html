Gables style guide for creating pages

Feature Blocks

This block will style a title, description and a box floated left. (Could contain an image, a video, etc.) The floated box will take up as much space as it needs for the content, so size your images accordingly. To float the box right instead of left, just add class 'right.'

Code


<div class="feature">
  <div class="thumb"><img alt="" src="http://placedog.com/222/137"></div>
  <p class="title">Lorem ipsum dolor sit amet</p>
  <p class="description">Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
  Phasellus elementum purus nec leo varius malesuada. Aenean commodo ante ac arcu mattis elementum. 
  Pellentesque felis nulla, sollicitudin id vulputate eget, tristique eget tellus. Fusce vitae nunc risus.</p>
</div>

Tabbed Content

If marked up correctly, this will provide a tabbed nav for portions of content that will become visible when their respective nav item is active.

The Code

The Tab Nav


<ul class="tab_nav">
  <li class="active"><a href="#tab_one_id">Tab One</a></li>
  <li class=""><a href="#tab_two_id">Tab Two</a></li>
  <li class=""><a href="#tab_three_id">Tab Three</a></li>
</ul>

The tab you want highlighted should have the class of 'active' as shown above.

The Tab Content


<div class="tab" id="tab_one_id">
  Tab One Content Goes Here
</div>
<div class="tab" id="tab_two_id">
  Tab Two Content Goes Here
</div>
<div class="tab" id="tab_three_id">
  Tab Three Content Goes Here
</div>

As you can see the id of each tab corresponds to an href for the links above.
